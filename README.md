# Pw Skills Website

### Setup instruction

1. Clone the project
```
    git clone https://github.com/pw-skills/pw-skills-website.git
    cd pw-skills-website
    npm install

```
2. Move into the directory

```
cd pw-skills-website

```

3. install dependancies
```
npm install

```

4. Run the project
```
    npm run dev

```
